# this.is.ryad.portfolio 🚀

Un portfolio moderne, fluide et performant conçu pour présenter mes projets, mes compétences et mes réalisations en tant que développeur **Full-Stack**.

Le design est épuré, adoptant une esthétique sombre (*dark mode*) haut de gamme avec des dégradés de rouge vif (`#ff0000`) et de rouge sombre (`#8b0000`), sublimé par des effets de flou (*glassmorphism*), de lueur (*card glow*) et des micro-animations interactives.

---

## 🎨 Caractéristiques & Design

- **Aesthétiques premium** : Couleurs sombres contrastées par une couleur d'accent rouge dynamique avec des effets de halo lumineux.
- **Animations fluides** : Effets de flottement, transitions douces et apparition des éléments au défilement (*Scroll Reveal*) grâce à l'API `IntersectionObserver`.
- **Structure propre & modulaire** :
  - `index.html` : Structure HTML5 sémantique et propre.
  - `style.css` : Styles CSS personnalisés (barre de défilement, grille de points en arrière-plan, etc.).
  - `main.js` : Comportement dynamique (menu mobile, défilement fluide, reveal d'éléments).
  - `tailwind.config.js` : Configuration thématique externe de Tailwind CSS.
- **Totalement responsive** : Conçu pour s'adapter parfaitement sur mobile, tablette et écran large.

---

## 🛠️ Technologies Utilisées pour le Portfolio

- **Structure** : HTML5 sémantique (SEO optimisé).
- **Stylisation** : [Tailwind CSS](https://tailwindcss.com/) (CDN) avec configuration personnalisée + CSS3 pur.
- **Logique & Animations** : JavaScript moderne (ES6+ / Vanilla JS).
- **Polices** : Google Fonts (*Inter* & *JetBrains Mono*).

## 📂 Projets Présentés

Le portfolio met en valeur quatre projets clés :

1. **DAWN&SEA — v1** : Plateforme touristique pour l'Algérie permettant de découvrir des destinations et hébergements.
   - *Stack* : Laravel, Tailwind CSS, MySQL, Blade.
2. **DAWN&SEA — v2 — IA** : Version robuste de la plateforme touristique avec un moteur de recommandation basé sur l'IA et l'analyse de préférences utilisateur.
   - *Stack* : Laravel, Tailwind CSS, Python (IA/ML), PostgreSQL, Supabase.
3. **Decision Making System — Book Store** : Système d'aide à la décision pour optimiser la gestion et les stocks d'une librairie.
   - *Stack* : Python, Data Analysis, Decision System.
4. **TP GridSim — Tri Distribué** : Simulateur de tri parallèle et d'ordonnancement de tâches sur une grille de calcul.
   - *Stack* : Java, GridSim, Calcul Distribué.

## 🚀 Lancement Local

Pour visualiser le portfolio en local sur votre machine :

1. **Cloner le dépôt** :
   ```bash
   git clone https://github.com/RYAD-BENYAKOUB/this.is.ryad.portfolio.git
   cd this.is.ryad.portfolio
   ```

2. **Ouvrir le projet** :
   - Ouvrez simplement le fichier `index.html` dans n'importe quel navigateur moderne.
   - Ou lancez un serveur de développement local (par exemple avec l'extension VS Code *Live Server* ou `npx serve .`).
